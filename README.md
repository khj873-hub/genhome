## 🗂️ 데이터 모델

### `inquiries` 테이블 (문의 폼 제출 데이터)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 식별자 |
| name | text | 신청자 이름 |
| phone | text | 연락처 |
| email | text | 이메일 주소 |
| service | text | 관심 서비스 (gov/ai/startup/ebook/other) |
| message | rich_text | 문의 내용 |
| submitted_at | datetime | 신청 일시 |

**API 엔드포인트**: `POST tables/inquiries`

## 🖼️ 이미지 교체 가이드

### 1. 히어로 배너 이미지 (황금열쇠 + 강사님 사진)
`index.html` 내 `hero-bg-placeholder` 영역에서:
```html
<!-- 기존 주석 제거 후 아래 태그 추가 -->
<img id="hero-banner-img" src="이미지URL" alt="배너" />
```

### 2. 강사 프로필 사진
`profile-img-placeholder` 내 `profile-avatar` div를 아래로 교체:
```html
<img src="강사사진URL" alt="강사 프로필" class="profile-img" />
```

### 3. 강연 사진
각 `.lecture-thumb` 내 주석 처리된 `<img>` 태그 활성화
